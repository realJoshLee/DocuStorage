We have many different pages to help you sort your tasks and to keep your life organized.

The Inbox page is for all of the tasks that you don't want to sort into a folder or that you don't want to set a due date for yet. Tasks will only show up in the Inbox and not put in there if they don't have a folder assigned.

The Planning page is the page where you can view all of the tasks that you planned. You have a 31 day view from the planning page.

The Logbook page is a log of all of the tasks that you completed. Under the task you can find the date that the task was completed. Tasks are sorted by the date that you completed them in. Recently completed to not recently completed. (Your admin can disable this is he/she find necessary. By default it's enabled.)

The Links page contains all of the saved website URLs that have been captured from the Google Chrome Extension. These tasks can't be moved to other places or have a due date set to them. To capture a task from the extension, (Guessing that you already have the extension setup.) click the save link button below the link that's shown.

The Stats page is the stats of your completed tasks. In this page you can view how many tasks you completed the current day by the hour, how many tasks you completed in the past 7 days, and how many tasks you complete each month.

The Add Folder button will make a popup show under your list of folder and you can type the folder that you would like to add.

The Settings page has all of the controls for your account. In this page you can do a lot. This page is the main control hub of everything that pertains to your account.

Adding tasks

Adding tasks is simple. On the left hand navigation panel, click on one of sections or folders then click where you see the text 'Add a task', start typing, then click on enter. This will seamlessly add a task. You can add tasks for up to 31 days ahead of the current date.


Completing task

To mark a task as complete, click in the grey box with curved corners to mark it as complete. This will remove the task from your view.


Subtasks

If there are uncompleted subtasks in a parent task, the icon highlighted in red will be shown.


Editing tasks

To edit a task, click on the text of the task that you want to edit. Once you do that, an overlay will show and you will have the option to edit everything. To edit content, click on where you want to edit and type. Once you update the task/content to your desire, click on the save button. To save a subtask (Also viewable from the same overlay), just click on the text and type. There is no save button for subtasks.


Due dates

When updating the due date for a task, type the new due date in the following format: M-DD-YYYY (Mar-05-2021) if you would like to add a task to the inbox, type inbox instead of the date.

If you would like to change the due date, you can drag and drop the task to different "date sections". To drag and drop click and hold on the text of the task, and drag it to the bottom of the list of tasks of the date that you would like to change it to. You will know that the drag and drop has been iniciated with a slight color backgound change will happened behind the tasks.


Sharing tasks

Follow the steps below to share a task with someone. When you share a task with someone, it can't be edited by the person it was shared with.

1. Click on the text of the task that you would like to share.

2. Click Notes/Sharing to change the page.

3. Click on the Enable/Disable Share button based on what you're doing.

4. Copy the link below and send it to the people you would like to share it with. When you share a task, the task content is viewable, notes, duedate, and subtasks. If a task is completed it is still viewable.


Add notes to a task

Follow the steps below to add notes to a task.

1. Click on the text of the task that you would like to add notes to.

2. Click Notes/Sharing to change the page.

3. Type your notes in the Notes section and click "Save Notes" to save the content.


Add a reminder to a task

Follow the steps below to add a reminder to a task. (Reminders can be enabled or disabled by your admin. Even if they are disabled you can set a value for this but you won't recieve a notification.) When you first load the app, you will get an email for each task that has a reminder set for that day.

1. Click on the text of the task that you would like to set a reminder for.

2. Click Notes/Sharing to change the page.

3. Type in the date for the reminder in the following format: M-DD-YYYY (Mar-05-2021)

To get to the admin panel, navigate to the Settings page and scroll all the way to the bottom. Next, click on Admin Dashboard/Management. Clicking this will open a new page.

When you first open the admin dashboard, you will see a bunch of sections with different stats and useful links to managing the deployment.

To manage all of the users who have signed up, in the Actions panel, click on Users. The user management page will load where you can select a user to manage.


To manage a certain user, look for their name and email and click on it. You will have some actions and features that you can view. From this panel, you can export all of their data (If enabled), set their preminum status, give them access to the admin panel, mark the user as active or suspend them, or verify them. Clicking the green buttons will put that feature/setting into effect and the red buttons will revoke that permission/settings. In this page you can also view the users full name, email, date the account was created, account id, their timezone, and their theme. If a user is suspeneded, they will not be able to log into their account, but it won't delete any of their data.

In the add user form, you can add a user. The default password for all users is Tasks1234! if you don't update it. When you add a user through this form, the whitelist feature will not take effect if you have it enabled and configured.

To manage all of the app settings, in the Actions panel, click on App Settings. The app management page will load and you will see a large amount of settings that can be configured to fine tune the app.

To revert all of the settings back to their default, click on the dropdown and select "Clear defaults" then click "Revert to install defaults".
