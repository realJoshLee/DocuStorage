App Install

This will walk you through on how to install our app. Our apps are based off of a PWA so that there isn't recompiling apps for a self hosted instance. The PWA app is supported for iOS and Android.

When you install the app as a PWA you don't have to worry about staying up to date. The app updates each time you load it.
