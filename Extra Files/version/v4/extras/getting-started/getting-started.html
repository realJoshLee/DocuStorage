Overview

Tasks is a modern task manager that will let you complete all of your tasks while being able to stay focused. Use Tasks to organize your life and to free up space in your brain.

Shortcuts

Pressing any of the following keys on a desktop expierence will compelte that shortcut.

ESC - Closes the task edit overlay.
ALT+1 - Opens inbox page.
ALT+2 - Opens planning page.
ALT+3 - Opens log page.
ALT+M - Cloes the nav bar on desktop.
ALT+B - Opens the nav bar on desktop.



Task management

Adding tasks

Adding tasks is simple. On the left hand navigation panel, click on one of sections or folders then click where you see the text 'Add a task', start typing, then click on enter. This will seamlessly add a task. You can add tasks for up to 31 days ahead of the current date.

Completing task

To mark a task as complete, click in the grey box with curved corners to mark it as complete. This will remove the task from your view. View the complete box in the image below.

Subtasks

If there are uncompleted subtasks in a parent task, the icon highlighted in red will be shown.

Editing tasks

To edit a task, click on the text of the task that you want to edit. Once you do that, an overlay will show and you will have the option to edit everything. To edit content, click on where you want to edit and type. Once you update the task/content to your desire, click on the save button. To save a subtask (Also viewable from the same overlay), just click on the text and type. There is no save button for subtasks.

Due dates

When updating the due date for a task, type the new due date in the following format: M-DD-YYYY (Mar-05-2021)

If you would like to change the due date, you can drag and drop the task to different "date sections". To drag and drop click and hold on the text of the task, and drag it to the bottom of the list of tasks of the date that you would like to change it to. You will know that the drag and drop has been iniciated with a slight color backgound change will happened behind the tasks.



Folders

When you have a task in a folder and scheule a date, it will not show up in the planning page nor will you be able to assign a task from the planning page into a folder.

Adding folders

To add a folder, click on the 'Add Folder' button in the bottom of the side nav, when that is clicked, a text box under the list of folders will appear. In that text box, type the name of the folder and then click enter to add the folder.



Editing folders

To edit a folder, click on the edit button to the right of the folder name, then a box will appear giving you the ability to edit a task. When you delete a folder, the folder will not be able to be recovered, but all tasks from that folder will be moved to the inbox.
